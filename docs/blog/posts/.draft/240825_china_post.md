# 中国邮政使用指南

写明信片和信多了，要收和想写的人也越来越多，一遍遍解释过于繁琐。遂修订之前本地写的指北文章《中国邮政寄信/明信片指北》，改正错误，补充内容，广纳意见与建议，形成更为完善的指南。

把「指北」改为「指南」是有意的。我以为「指北」比起「指南」的水平更低些，要用北的方向去推测南的方向。或者更具体些，「指北」更偏向于试错经验，而非「指南」的正确方法总结。

本文从寄出者的角度，按流程顺序描述国内信件的各环节。末尾对国际平常明信片稍加描述，以方便使用 Postcrossing。

## 概念与原则

推荐性国家标准[《邮政业术语》（GB/T 10757-2011）](https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=8E5F9EB579BADC1DE339289317FF327F)规定了部分术语的概念。为通俗但不致混淆，本文选择性采用了一部分。

- 信件：信函和明信片的统称。
- 给据：也称「挂号」
- 平常：简称「平」
- 处理流程
- 营业员：
- 投递员：

信件与快件尽管有相似之处，但从处理流程的角度看有诸多不同之处。基本原则：**不要用「快件不是这样」来对待信件。**

## 获取信息

首先需要有对方的姓名、电话、地址，直接要就行了。大体上，这和在京东淘宝填收货信息差不多，但地址有一些细节需要注意。

首先，地址一般需要按照「三级行政区划[^1]-路名-号码-名称-部门」填写。

例如，这个是我现在用的地址：重庆市北碚区天生路 2 号西南大学北碚校区茶园一舍。拆解开变成：

- 三级行政区划：重庆市北碚区
- 路名：天生路
- 号码：2 号
- 名称：西南大学北碚校区
- 部门：茶园一舍

三级行政区划中，区，如「北碚区」，不可省略；但社区街道，如「天生街道」，不属于这一部分，也不需要填写。某些平台在填写快件地址时要求填写街道，可能是为了方便投递，但此处并不需要。

此处所说的部门，是在名称内部更精细的划分方式。基于信件的投递方式，此处必须填写，而不能像快件地址一样省略。此处按住宅和学校两种进行说明。

如果名称填的是住宅，需要精确到门牌号，例如「X 号楼 Y 单元 ABC」。

如果名称填的是学校，需要精确到班级或宿舍楼，不同学校要求不同。通常而言，高等教育（大学）之前填写班级，之后按学校收发室要求填写，如果不知道要求，填写班级更安全。例如：

- 高三（20）班
- 农学与生物科技学院 2025 级生物育种 02 班
- 茶园一舍

另外，有隐私顾虑的可以接受假名，但要易读，方便寄出者书写、营业员录入和投递员打电话。基于此，不推荐使用英文、过长或易社死的假名。

## 购买道具

### 信封

如果要写很多字或者考虑隐私，建议使用信函。这时，你需要买信封。

如果你所在的学校有自己专有的信封，正面印了校名的那种，那是一个很好的选择，显得正式，并且可以省去写寄件地址的过程；但通常会更贵，且部分学校很难弄到。

如果没有，得力京东自营旗舰店可以买到普通的信封，且换算单个信封价格通常比晨光京东自营旗舰店更便宜。

最后，如果用量不大或者时间紧迫，邮政柜台也可以买到信封。

推荐性国家标准[《信封》（GB/T 1416-**2021**）](https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=DE308FCC68F31C0B7C6F169AA5BFA593)新增了对信封正面地址、收件人、电话三根书写线的要求，而旧版国标（2003 版）信封正面是空白的。部分厂商可能会把新旧国标的信封混卖，但两种正面布局都是合理的。

对于信封大小，同样可参照以上国标，但厂商出于未知原因，销售时通常会用「X 号」来代替标准规格代号。不管叫什么，都可以直接用规格去换算。对照表如下：

| 标准规格代号 | 数字代号 | 规格（mm） |
| :---: | :---: | :---: |
| B6 | 3 号 | 176x125 |
| DL | 5 号 | 220x110 |
| ZL | 6 号 | 230x120 |
| C5 | 7 号 | 229x162 |
| C4 | 9 号 | 324x229 |

经验而言，对于尺寸在 A4 纸及以下的内容物，最常见的 DL 信封已经足够。如果要寄更大只的，比如照片一类，可以考虑 C5 或 C4 信封，但能用上 C4 的场景很少。

### 明信片

如果只写几行字且无需考虑隐私问题，可以直接选用明信片。明信片的邮费比信函稍便宜。

某种程度上，明信片讲究图案面稀有性，以致于可以被当成艺术品对待。通过各种工艺，明信片的精美程度会大大提高，但制作成本也相应增加。

我个人很推崇自行定制明信片，推荐淘宝店铺「纸言片语」。定制需要提前拍摄和设计，但是效果很好，作礼物也很有感觉。

如果没有合适的图片，所在区域的明信片也是不错的选择。「所在区域」可以是旅游时遇到的景点，可以是学校，也可以是城市。

随便就能买到的当然也能用，世界名画、文化创意一类。淘宝的「创意明信片店铺榜」也可以逛逛。目前邮政管理不算严格，写了收件人信息贴了邮票都能寄得出去，所以书写面是白的没有邮编框，大概也无所谓。

无论怎么买，图案面总是要有一点审美的，如果是一个真实的地点，最好还能看出来到底是哪。对方看不出来的话，在书写面写上是一个不错的选择，算是给对方涨知识了。
