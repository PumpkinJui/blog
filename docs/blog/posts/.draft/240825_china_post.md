# 中国邮政使用指南

写明信片和信多了，要收和想写的人也越来越多，一遍遍解释过于繁琐。遂修订之前本地写的指北文章《中国邮政寄信/明信片指北》，改正错误，补充内容，广纳意见与建议，形成更为完善的指南。

把「指北」改为「指南」是有意的。我以为「指北」比起「指南」的水平更低些，要用北的方向去推测南的方向。或者更具体些，「指北」更偏向于试错经验，而非「指南」的正确方法总结。

1. 只能指个北，指南(包括怎么写信封和明信片)可以看：
   - [知乎：现如今怎样寄信？](https://www.zhihu.com/question/330393975/answer/1633942631)
   - [少数派：在一秒发送消息的时代，如何寄出一封手写信件](https://sspai.com/post/77691)
2. 邮局在哪以实地为准，基本别信邮政小程序，有时可以参考地图软件
3. 只有中国邮政可以寄信/明信片/EMS，中国邮政储蓄银行不能寄；可以看招牌和门侧面的牌子确认
4. EMS 需带身份证，其余一般不用
5. 某些邮政支局只收现金
6. 挂号有助于让信/明信片快点送到，还能给个(一般只有收藏用途的)凭证；代价是多付一块二。建议能挂号都挂号
7. 邮政大部分施行朝九晚五，以上时间到不了邮局可以投邮筒；但是邮筒必须贴好邮资再投，不能挂号，且不保证什么时候取件。建议能柜台就柜台
8. 写信封和明信片时候，给条码和邮票留点地方。即使不挂号也会有邮政内部的平信条码，另外邮票面额就那么几个，比如外埠挂号明信片三块八就没有单独的邮票，如果没有贴两张的地方就只能超额贴成一张四块二。
9. 邮资以柜台称重计算为准，可以提前在以下网站查询
   - [国家邮政局，国内普通邮件业务资费](https://www.spb.gov.cn/gjyzj/c200092/201612/da9c512599244df1a273b2174d420f4a.shtml)
   - [中国邮政，邮资查询电脑版](https://dey.11185.cn/web/#/jdpostage)
   - [中国邮政，邮资查询手机版](https://dey.11185.cn/wx/#/tariffQuery)
